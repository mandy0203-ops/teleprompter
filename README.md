# 提詞機 · Teleprompter

語音跟讀提詞機。仿影視颶風款式，零依賴、單檔 HTML。

## 線上使用

直接打開：**https://mandy0203-ops.github.io/teleprompter/**

任何裝置（Mac / Windows / iPad / iPhone）只要用 Chrome / Edge 開啟即可。第一次會跳麥克風授權，按「允許」一次永久有效。

## 功能

- **自動播放**：固定速度往下捲（傳統提詞機）
- **語音跟讀**：辨識你正在念的字，自動跟到那一行
- 速度 / 字級 / 行高 / 字色 / 背景色都可調
- 水平 / 垂直鏡像（給玻璃反射式提詞架用）
- 倒數 3-2-1 + 引導語（深呼吸 → 看向鏡頭 → 準備）
- 全螢幕 / 完成儀式 / 浮動麥克風指示器

## 鍵盤快捷

| 鍵 | 功能 |
|---|---|
| 空白 | 自動播放 / 暫停 |
| V | 切換語音跟讀 |
| ↑ ↓ | 調整速度 |
| R | 回到開頭 |
| F | 全螢幕 |
| Esc | 退出全螢幕 |

## 加到主畫面

- **iPhone / iPad**：Safari 分享 → 加到主畫面
- **Mac / Windows Chrome**：網址列右邊「安裝」icon → 變獨立 app

## 注意事項

- 必須用 **Chrome 或 Edge**（Safari 對連續語音識別支援有限）
- 語音辨識需要網路連線（Chrome 雲端辨識）
- 你的稿件存在你瀏覽器的 localStorage，origin 隔離——別人看不到

## 跟丟率

中文一般語速、辨識準確率正常的條件下，跟讀對齊成功率約 75–80%。對不上時按 R 從目前位置重對齊即可。
